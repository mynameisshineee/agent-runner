# Contributing

## Development

- Runtime: Bun >= 1.2
- Keep terminal-first semantics as default for BYO mode.
- Never commit secrets/tokens.

## Principles

- PM is source of truth for jobs and status.
- Do not mark jobs as running without real process/session start.
- Keep adapter interfaces stable and provider-agnostic.

## Pull requests

- Include a short test transcript for changed flows.
- Update README when adding endpoints/flags.
